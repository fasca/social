Social Network
==============

A simple social network.

Installation
------------

Will need NodeJs to run.
Will use ExpressJs Framework (Server-Side)

Clone the repo :

~~~sh
git clone https://github.com/fasca/social
~~~

Install the dependencies :
~~~sh
npm install
~~~

Run the server :
~~~sh
node app
~~~

The website will run at localhost:3000
